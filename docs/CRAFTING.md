# GOHENCRAFT - Sistema de Crafting e Receitas

## 🔧 Mecânicas de Crafting

### Tipos de Crafting

#### 1. Crafting Manual (Inventário 2x2)
Receitas simples que podem ser feitas no inventário do jogador.

#### 2. Crafting Table (3x3)
Receitas mais complexas que requerem uma mesa de crafting.

#### 3. Fornalha (Smelting)
Transformação de recursos através do calor.

#### 4. Crafting Elemental (Futuro)
Crafting especial usando poderes elementais.

## ⚡ Crafting Elementais (Sistema Avançado)

### Mecânica

Disponível após escolher um Caminho de Poder.

Permite criar itens especiais usando essência do elemento escolhido.

### Caminho da Água - Receitas

#### Frasco de Água Purificada
```
Crafting Elemental

[Vidro] + [Essência de Água] = Frasco de Água Purificada
```
**Efeito:** Restaura 50 HP instantaneamente

#### Armadura Aquática
```
Crafting Table + Essência de Água

[Essência] [     ] [Essência]
[Essência] [Armadura] [Essência]
[Essência] [     ] [Essência]
```
**Efeito:** +50% velocidade de nado, respiração aquática

## 📈 Progressão de Crafting

### Tier 0: Inicial
- Ferramentas de madeira
- Crafting table
- Abrigo básico

### Tier 1: Pedra
- Ferramentas de pedra
- Fornalha
- Mineração básica

### Tier 2: Metais
- Ferramentas de cobre/ferro
- Armadura básica
- Armas melhoradas

### Tier 3: Elemental
- Crafting elemental desbloqueado
- Itens com efeitos especiais
- Armaduras elementais

### Tier 4: Divino
- Crafting com essências de boss
- Equipamentos divinos
- Itens únicos
